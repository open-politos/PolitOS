# Membership

Membership tiers, onboarding flows, and rights matrix.

## Default Tiers

| Tier | Rights | Requirements |
|---|---|---|
| Observer | Read, Comment | Registration |
| Member | Read, Comment, Propose, Vote | Registration + acceptance of principles |
| Verified Member | All above + Panel eligibility | Identity verification |

## Configuration

Membership tiers and rights are configured in `config/party.config.yaml`. The defaults above can be adjusted per organization.

## Files

- `rights-matrix.yaml` — Detailed mapping of rights per tier
- `onboarding.yaml` — Steps for new member registration
